\# Тема приложения: Мини-Словарь



\## Описание

Мобильное приложение для поиска определений английских слов.



\## Основные функции

\- Ввод слова в текстовое поле

\- Нажатие кнопки «Найти»

\- Отображение:

&nbsp; - введенное слово,

&nbsp; - часть речи слова,

&nbsp; - определение слова,

&nbsp; - примера использования слова (если такое в апишке будет)

\- Сохранение каждого успешно найденного слова в отдельном экране \*\*История\*\*

\- Просмотр истории на отдельном экране

\- Удаление слов из истории



\## Эскизы интерфейса

!\[Экран «История»](images/history\_screen.png)



1\. \*\*Главная\*\*

&nbsp;  - Поле ввода для слова

&nbsp;  - Кнопка «Найти»

&nbsp;  - Область вывода результата:

&nbsp;    - Слово 

&nbsp;    - Часть речи

&nbsp;    - Определение

&nbsp;    - Пример (если такое в апишке будет)



2\. \*\*История\*\*

&nbsp;  - Список ранее найденных слов (только слова, без определений)

&nbsp;  - Нажатие на слово возврат на главный экран и автоматический поиск этого слова

&nbsp;  - Иконка удаления рядом с каждым словом



\## Навигация

\- Главный экран - кнопка «Главная»

\- История ранее введенных слов - кнопка «История»

\- Нажатие на слово в истории - возврат на главный экран и начало поиска



\## Сценарии использования

1\. \*\*Поиск слова\*\* 

\- Пользователь Вводит слово

\- Нажимает «Найти»

\- Получает определение 

\- Слово сохраняется в историю



2\. \*\*Просмотр истории\*\*  

\- Пользователь открывает экран истории 

\- Высвечивается список ранее вводимых слов

\- Пользователь может повторно открыть нужное слово

\- Пользователь может удалить любое слово
